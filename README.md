# Panex_Privus

This will contain python code and work flow notes to develop VCF, GFA, XMFA from multi-genome (pan-genome) collections and find variant sites (SNP, INDEL, SV) present in target genome group A and NOT present in non-target genome group B (e.g. target group A all have the same variant at site X and non-target group B do not have that variant).
